## Contributor License Agreement (CLA)
In order to contribute to Lunogram project, you may need to sign a CLA. 
You can find the [CLA document here](https://github.com/lunogram/cla/blob/main/CLA.md).

### How to sign
Upon open the pull request to any of the project under Lunogram organisation, a bot
will assess whether you have signed the CLA or not, if not it will comment on pull rquest
with instruction to sign it.
